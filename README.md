
# O que esse projeto é: 

Esse projeto são anotações feitas durantes as aulas de GIT e GitHub na @DigitalInnovationOne em collab com @leonardocs para estudos e aprendizados.

# O que contém nos arquivos ?

Dentro dos arquivos possuí uma página em HTML, contendo todas anotações sobre as aulas, desde coceitos básicos, códigos, configurações etc e para quem preferir também à um arquivo em PDF com as mesmas caracteristicas da páginas.

# Processo de criação

A criação do conteúdo foi em conjunto com Leonardo, a partir do site Notion, onde fizemos um projeto em conjunto, realizando anotações das aulas que poderia ser útil para nós e para outras pessoas no futuro, com intuito de ajudar nos estudos e melhor absorção de conhecimento. 

### HTML
A página em HTML foi uma opção do próprio Notion para exportar todo conteúdo criado dentro dele em uma página HTML. 

### PDF 
O conteúdo em PDF foi gerado do mesmo modo que foi gerado a página em HTML, por uma função do próprio Notion para transformar toda as anotação em um arquivo PDF
